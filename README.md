# Tobiloba Akala - Personal Portfolio 🚀

A sleek, high-performance personal portfolio website built to showcase my projects, skills, experience, and technical blog. Designed with a focus on clean code, accessibility, and high-end visual aesthetics using modern web technologies.

![Portfolio Preview](https://raw.githubusercontent.com/soumyajit4419/Portfolio/master/src/Assets/home-main.svg)

## ✨ Features

* **Modern UI/UX:** Crafted with a stunning Glassmorphism aesthetic and a dark-themed, neon-accented color palette.
* **Interactive Animations:** Features smooth page transitions, hover effects, and an interactive particle background using Framer Motion and tsParticles.
* **Fully Responsive:** Mobile-first design ensuring a seamless experience across all devices and screen sizes.
* **Dynamic Typing Effect:** Engaging hero section with a customized typewriter effect.
* **Blog Section:** Integrated blog layout to share articles, tutorials, and insights on frontend engineering.
* **Resume Integration:** Direct download link for my latest CV/Resume.

## 🛠️ Tech Stack

This project was built using the following technologies:

* **Framework:** [React.js](https://reactjs.org/) (Functional Components & Hooks)
* **Language:** [TypeScript](https://www.typescriptlang.org/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Routing:** [React Router v6](https://reactrouter.com/)
* **Animations:** [Framer Motion](https://www.framer.com/motion/) & [react-tsparticles](https://particles.js.org/)
* **Icons:** [React Icons](https://react-icons.github.io/react-icons/)
* **Build Tool:** [Vite](https://vitejs.dev/)
