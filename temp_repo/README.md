# TOBILOBADEV / PORTFOLIO v1.0

A high-performance, dark-themed developer portfolio built with **React 18** and **Tailwind CSS**. This project serves as a centralized hub for my frontend engineering work and technical identity.

## 📱 The "Mobile-First" Engineering Feat
This repository is unique: **it was developed and deployed entirely from a mobile device.** From component architecture to debugging complex syntax errors and managing Vercel deployment pipelines, every line of code was written without a desktop environment. It stands as a testament to technical resourcefulness and a "no-excuses" approach to shipping software.

---

## 🛠️ Tech Stack

* **Framework:** React 18 (Vite)
* **Styling:** Tailwind CSS (Utility-first architecture)
* **Icons:** Lucide React
* **Deployment:** Vercel (CI/CD Pipeline)
* **Type:** Module-based JavaScript

---

## 🚀 Key Features

* **Data-Driven UI:** Project cards and social links are managed via a centralized `data.js` for easy updates.
* **Glassmorphism Design:** Modern backdrop-blur navigation and subtle border treatments for a premium aesthetic.
* **Responsive Engine:** Fully optimized for high-end desktop displays down to mobile viewports.
* **Asset Pipeline:** Integrated handling for professional `.jpg` thumbnails and `.pdf` documentation.
